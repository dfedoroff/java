## Задача 2

### Описание:

Написать метод, принимающий на вход два целых числа `n`, `m` и проверяющий, что их сумма лежит в пределах от `10` до `20` (включительно), если да, вернуть `true`, в противном случае вернуть `false`.

### Примеры:

```
Ввод: 7
      8
Вывод: true
```
```
Ввод: 4
      1
Вывод: false
```
