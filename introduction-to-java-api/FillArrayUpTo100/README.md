## Задача 8

### Описание:

Задать пустой целочисленный массив длиной `100`. С помощью цикла заполнить его значениями `1 2 3 4 5 6 7 8 … 100`.

### Пример:

```
Ввод: 100
Вывод: [1, 2, 3, 4, 5, 6, 7, 8 ... 100]
```
