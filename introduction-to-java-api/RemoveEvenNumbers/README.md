## Задача 1

### Описание:

Пусть дан произвольный список целых чисел, удалить из него четные числа.

### Пример:

```
Ввод: [8, 2, 9, 4, 7, 4, 7, 2, 2, 8]
Вывод: [9, 7, 7]
```
