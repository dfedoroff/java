## Задача 12

### Описание:

Задать одномерный массив размера `n` и найти в нем минимальный и максимальный элементы.

### Пример:

```
Ввод: [-13, 20, 98, 87, 97, 12, 53, -95, 68, -10]
Вывод: -95
        98
```
