## Задача 4

### Описание:

Написать метод, которому в качестве параметра передается целое число `n`. Метод должен вернуть `true`, если число отрицательное, и вернуть `false` если положительное.

### Примеры:

```
Ввод: -7
Вывод: true
```
```
Ввод: 5
Вывод: false
```
