## Задача 10

### Описание:

Создать двумерный целочисленный массив с одинаковым количеством `n` строк и столбцов. Используя цикл присвоить всем диагональным элементам двумерного массива значение `1`.

### Пример:

```
Ввод: 3
Вывод:
[1, 0, 1]
[0, 1, 0]
[1, 0, 1]
```
