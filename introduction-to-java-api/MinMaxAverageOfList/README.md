## Задача 2

### Описание:

Задан целочисленный список `ArrayList`. Найти минимальное, максимальное и среднее арифметическое значение этого списка.

### Пример:

```
Ввод: [2, 3, 9, 4, 5, 5, 1, 7, 7, 3]
Вывод: 1
       9
       4.6
```
