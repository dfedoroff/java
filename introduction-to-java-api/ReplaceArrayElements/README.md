## Задача 7

### Описание:

Задать целочисленный массив, состоящий из элементов `0` и `1`. С помощью цикла и условия заменить `0` на `1`, `1` на `0`.

### Пример:

```
Ввод: [1, 1, 0, 0, 1, 0, 1, 1, 0, 0]
Вывод: [0, 0, 1, 1, 0, 1, 0, 0, 1, 1]
```
