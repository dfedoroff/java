## Задача 9

### Описание:

Задать массив `{  1, 5, 3, 2, 11, 4, 5, 2, 4, 8, 9, 1 }` пройти по нему циклом, и числа меньшие `6` умножить на `2`.

### Пример:

```
Ввод: [1, 5, 3, 2, 11, 4, 5, 2, 4, 8, 9, 1]
Вывод: [2, 10, 6, 4, 11, 8, 10, 4, 8, 8, 9, 2]
```
